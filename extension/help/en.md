# IPA Reader - Help

## Getting Started
1. Install IPA Reader from the Chrome Web Store
2. Visit any English web page
3. IPA pronunciation annotations will appear above English words

## Features
- **Enable/Disable**: Toggle the extension on/off from the popup
- **Whole Page IPA**: Show IPA for all words at once
- **Accent Type**: Switch between American and British English
- **Hover Tooltip**: Hover over annotated words to see IPA in a tooltip
- **Text-to-Speech**: Click the speaker button or select text to hear pronunciation
- **Speech Rate**: Adjust reading speed in settings
