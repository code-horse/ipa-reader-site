# IPA Reader - 사용자 가이드

> 버전: v1.3.0

## 소개

IPA Reader는 영어 학습자를 위해 설계된 브라우저 확장 프로그램입니다. 웹 페이지의 영어 단어 옆에 IPA(국제 음성 기호) 발음 주석을 추가하며, 미국식과 영국식 영어 악센트를 모두 지원하여 영어 발음을 더 쉽게 학습할 수 있도록 도와줍니다.

---

## 주요 기능

- **전체 페이지 IPA 모드** — 한 번의 클릭으로 페이지의 모든 영어 단어에 IPA 주석 추가, 색상으로 구분된 IPA 기호(모음, 자음, 강세 표시)로 쉽게 읽기
- **약형/강형 표시** — 기능어(the, to, can 등)의 약형 및 강형 발음 변형을 자동으로 표시하여 자연스러운 연결 발화 습득을 지원
- **미국식 & 영국식 악센트** — 미국식 영어(en-US)와 영국식 영어(en-GB) IPA 간 전환 가능
- **텍스트 음성 변환** — 스피커 버튼을 클릭하여 선택한 악센트에 맞는 발음을 들을 수 있습니다
- **카루오케 효과가 있는 선택 영역 읽기** — 영어 텍스트를 선택하면 읽기·번역 버튼이 있는 작은 도구 모음이 나타납니다. 음성과 동기화된 실시간 단어별 강조(카루오케 효과)로 재생됩니다
- **선택 영역 번역** — 아무 텍스트나 선택한 뒤 도구 모음의 번역 버튼을 누르면 Bing 또는 Google 번역으로 즉시 번역되며 인라인 말풍선에 표시됩니다
- **호버 툴팁** — 주석이 달린 단어 위에 마우스를 올리면 IPA와 발음 버튼 표시
- **동형이음어 식별** — 여러 발음을 가진 단어(read, live 등)를 자동으로 식별하고 문맥에 따라 올바른 발음을 선택
- **키보드 단축키** — 사용자 지정 가능한 단축키로 핵심 기능에 빠르게 액세스
- **다국어 인터페이스** — 38개 인터페이스 언어 지원

## 사용 방법

### 1단계: 확장 프로그램 설치

[Chrome Web Store](https://chromewebstore.google.com/)에서 **IPA Reader**를 설치하거나, 개발자 모드에서 로컬로 로드하세요.

### 2단계: 웹 페이지 열기

영어 콘텐츠가 포함된 웹 페이지를 방문하세요.

### 3단계: IPA 켜기

브라우저 도구 모음의 확장 프로그램 아이콘을 클릭합니다. "IPA 활성화"를 켠 다음 "전체 페이지 IPA"를 켜면 페이지의 모든 단어에 주석이 붙습니다. 오른쪽 하단의 플로팅 버튼으로도 전환할 수 있습니다.

### 4단계: IPA 보기

단어 위에 마우스를 올리면 IPA 툴팁이 표시되고, 스피커 아이콘을 클릭하면 발음을 들을 수 있습니다. 약형/강형이 있는 기능어는 툴팁에 두 가지 변형이 모두 표시됩니다.

### 5단계: 선택한 텍스트 읽기 및 번역

마우스로 영어 텍스트를 선택합니다. 선택 영역 근처에 작은 도구 모음이 나타나며 두 개의 버튼이 있습니다.
- **🔊 읽기** — 선택한 텍스트를 읽어 주며 카루오케처럼 단어별로 강조합니다
- **🌐 번역** — 도구 모음 아래에 인라인 번역 말풍선을 표시합니다

오른쪽 클릭 메뉴에서 "IPA Reader > Speak Selection" 또는 "IPA Reader > Translate Selection"을 선택할 수도 있습니다.

> **팁:** 브라우저 도구 모음의 확장 프로그램 아이콘을 클릭하면 설정 패널이 열려 악센트 유형, 읽기 속도, 번역 엔진 등을 조정할 수 있습니다.

## 전체 페이지 IPA 모드

전체 페이지 IPA 모드를 켜면 페이지의 각 영어 단어 위에 루비 형식으로 IPA가 표시됩니다. IPA 기호는 유형별로 색이 구분됩니다.

- **모음** — 파란색 강조
- **자음** — 회색 표시
- **강세 표시**(ˈ ˌ) — 빨간색 강조
- **장음 표시**(ː) — 보라색 강조

확장 프로그램은 줄 높이를 자동으로 조정하고 주석이 본문과 겹치지 않게 하며, 단어 길이에 따라 IPA 글꼴 크기를 조절합니다.

---

## 약형/강형 표시

영어의 많은 기능어에는 두 가지 발음이 있습니다.

- **약형** — 자연스러운 연결 발화에서 쓰이는 축약 발음(예: the → /ðə/)
- **강형** — 강조나 단독 사용 시의 완전한 발음(예: the → /ðiː/)

"약형/강형 표시"가 켜져 있으면 이러한 단어에 마우스를 올리면 툴팁에 "WEAK"와 "STRONG" 레이블이 붙은 두 변형이 함께 표시됩니다.

포함되는 단어: 관사(the, a, an), 전치사(to, for, of, from, at, as, than), 접속사(and, or, but), 대명사(you, your, her, him, his, them, us, our, there), 조동사(am, is, are, was, were, been, can, could, would, should, must, shall, will, do, does, have, has, had) 등입니다.

## 선택 영역 읽기 및 카루오케

선택 영역 읽기로 영어 텍스트를 선택한 뒤 한 번의 클릭으로 읽을 수 있습니다. 문장 발음 학습과 읽기 연습에 적합합니다.

**방법 1: 선택 도구 모음**  
마우스로 영어 텍스트를 선택합니다. 선택 영역 근처에 🔊 읽기와 🌐 번역 버튼이 있는 작은 도구 모음이 나타납니다. 읽기를 클릭하여 재생합니다. 읽는 동안 각 단어가 실시간으로 강조됩니다(카루오케 효과).

**방법 2: 오른쪽 클릭 메뉴**  
영어 텍스트를 선택한 후 오른쪽 클릭하여 "IPA Reader > Speak Selection"을 선택하세요.

**방법 3: 키보드 단축키**  
텍스트를 선택한 뒤 `Alt+Shift+S`(Mac: `Ctrl+Shift+S`)를 누릅니다.

> **팁:** 카루오케 강조는 브라우저가 TTS 단어 경계 이벤트를 지원할 때 가장 잘 작동합니다. 지원하지 않으면 확장 프로그램이 시간 기반 대체 방식으로 부드럽게 강조합니다.

## 번역

페이지에서 아무 텍스트나 선택하여 번역 기능으로 즉시 번역을 받을 수 있습니다.

**방법 1: 선택 도구 모음**  
텍스트를 선택한 뒤 도구 모음의 🌐 번역을 클릭합니다. 결과가 도구 모음 아래 말풍선에 표시되며 복사 버튼이 있습니다.

**방법 2: 오른쪽 클릭 메뉴**  
텍스트를 선택하고 오른쪽 클릭 후 "IPA Reader > Translate Selection"을 선택하세요.

**방법 3: 키보드 단축키**  
텍스트를 선택한 뒤 `Alt+Shift+T`(Mac: `Ctrl+Shift+T`)를 누릅니다.

**번역 엔진:**
- **Bing 번역**(기본값) — Microsoft Translator 제공
- **Google 번역** — Google 제공

두 엔진 모두 **108개 대상 언어**를 지원합니다.

확장 프로그램 설정에서 번역 엔진과 대상 언어를 바꿀 수 있습니다. 대상 언어는 브라우저 언어에서 자동으로 감지됩니다.

> **팁:** 도구 모음이나 말풍선 바깥을 클릭하면 닫힙니다.

## 키보드 단축키

| 단축키 | Mac 단축키 | 동작 |
|----------|-------------|--------|
| `Alt+Shift+I` | `Ctrl+Shift+I` | IPA 주석 켜기/끄기 |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 선택 영역 읽기 |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 선택 영역 번역 |

> **팁:** Chrome의 `chrome://extensions/shortcuts`에서 이 단축키를 사용자 지정할 수 있습니다.

## 자주 묻는 질문

**Q: 일부 페이지에서 작동하지 않는 이유는?**  
A: 보안상의 이유로 브라우저 확장 프로그램은 `chrome://`, 브라우저 설정, Chrome Web Store 등 특수 페이지에서 실행할 수 없습니다.

**Q: 일부 단어에 IPA가 없는 경우는?**  
A: IPA 사전은 일반적인 영어 단어를 다룹니다. 사전에 없는 단어의 경우, 확장 프로그램이 레마타이제이션 및 G2P를 통해 근사 IPA를 생성하며, 툴팁에 ≈ 또는 ~로 표시됩니다.

**Q: 텍스트 음성 변환에 소리가 없어요?**  
A: 시스템 볼륨 설정을 확인하고 영어 음성 팩이 설치되어 있는지 확인해 주세요. 브라우저와 운영 체제에 따라 음성 지원이 다릅니다.

**Q: 전체 페이지 모드가 레이아웃에 영향을 주나요?**  
A: IPA 주석에는 추가 공간이 필요합니다. 확장 프로그램이 줄 높이를 조정해 영향을 줄입니다. 읽기에 불편하면 전체 페이지 모드를 끄고 호버 툴팁을 사용하세요.

**Q: 툴팁의 ~ 및 ≈ 기호는 무엇을 의미하나요?**  
A: ~는 IPA가 규칙 기반 변환(G2P)으로 생성되었음을, ≈는 관련 기본 단어에서 파생되었음을 나타냅니다. 사전 항목보다 정확도가 낮을 수 있습니다.

**Q: 번역이 작동하지 않아요.**  
A: 번역에는 인터넷 연결이 필요합니다. Bing 번역이 실패하면 설정에서 Google 번역으로 바꿔 보세요. 일부 네트워크 환경에서는 번역 서비스에 접근할 수 없을 수 있습니다.
