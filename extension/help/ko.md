# IPA Reader - 사용자 가이드

> 버전: v1.1.0

## 소개

IPA Reader는 영어 학습자를 위해 설계된 브라우저 확장 프로그램입니다. 웹 페이지의 영어 단어 옆에 IPA(국제 음성 기호) 발음 주석을 추가하며, 미국식과 영국식 영어 악센트를 모두 지원하여 영어 발음을 더 쉽게 학습할 수 있도록 도와줍니다.

---

## 주요 기능

- **텍스트 선택 주석** — 웹 페이지에서 영어 텍스트를 선택하면 IPA와 음성 버튼이 자동으로 표시됩니다
- **전체 페이지 IPA 모드** — 한 번의 클릭으로 페이지의 모든 영어 단어에 IPA 주석 추가, 색상으로 구분된 IPA 기호(모음, 자음, 강세 표시)로 쉽게 읽기
- **미국식 & 영국식 악센트** — 미국식 영어(en-US)와 영국식 영어(en-GB) IPA 간 전환 가능
- **텍스트 음성 변환** — 스피커 버튼을 클릭하여 선택한 악센트에 맞는 발음을 들을 수 있습니다
- **선택 영역 읽기** — 영어 텍스트를 선택하고, 플로팅 버튼을 클릭하거나 오른쪽 클릭 후 "선택 영역 읽기"를 선택하면 재생됩니다
- **호버 툴팁** — 주석이 달린 단어 위에 마우스를 올리면 IPA와 발음 버튼 표시
- **약형/강형 표시** — 기능어(the, to, can 등)의 약형 및 강형 발음 변형을 자동으로 표시하여 자연스러운 연결 발화 습득을 지원
- **동형이음어 식별** — 여러 발음을 가진 단어(read, live 등)를 자동으로 식별하고 문맥에 따라 올바른 발음을 선택
- **다국어 인터페이스** — 38개 인터페이스 언어 지원

## 사용 방법

### 1단계: 확장 프로그램 설치

[Chrome Web Store](https://chromewebstore.google.com/)에서 **IPA Reader**를 설치하거나, 개발자 모드에서 로컬로 로드하세요.

### 2단계: 웹 페이지 열기

영어 콘텐츠가 포함된 웹 페이지를 방문하세요.

### 3단계: 텍스트 선택 또는 플로팅 버튼 사용

주석을 추가할 영어 텍스트를 선택하거나, 오른쪽 하단의 플로팅 버튼을 클릭하여 전체 페이지 IPA 모드를 활성화하세요.

### 4단계: IPA 보기

단어 위에 마우스를 올리면 IPA 툴팁이 표시되고, 스피커 아이콘을 클릭하면 발음을 들을 수 있습니다.

### 5단계: 선택한 텍스트 읽기

마우스로 영어 텍스트를 선택한 후 🔊 플로팅 버튼을 클릭하여 읽거나, 오른쪽 클릭 후 "선택 영역 읽기"를 선택하세요.

> **팁:** 브라우저 도구 모음의 확장 프로그램 아이콘을 클릭하면 설정 패널이 열려 악센트 유형, 읽기 속도 등을 조정할 수 있습니다.

---

## 선택 영역 읽기

선택 영역 읽기 기능은 영어 텍스트를 선택한 후 한 번의 클릭으로 읽을 수 있게 해주며, 문장 발음 학습과 읽기 연습에 완벽합니다.

**방법 1: 플로팅 버튼**  
마우스로 영어 텍스트를 선택하면 선택 영역 오른쪽 상단에 스피커 버튼이 나타납니다 — 클릭하여 읽기.

**방법 2: 오른쪽 클릭 메뉴**  
영어 텍스트를 선택한 후, 오른쪽 클릭하여 메뉴에서 "선택 영역 읽기"를 선택하세요.

> **팁:** 선택 영역 읽기는 설정에서 구성한 문장 읽기 속도를 사용합니다. 단어 발음은 기본 속도를 사용합니다. TTS 음성은 선택한 악센트 유형(미국식 또는 영국식)과 일치합니다.

## 설정 가이드

| 설정 | 설명 |
|------|------|
| **IPA 활성화** | IPA 주석 기능을 켜거나 끄는 마스터 스위치 |
| **전체 페이지 IPA** | 활성화하면 페이지의 모든 영어 단어에 IPA 표시(페이지 레이아웃에 영향할 수 있음) |
| **악센트 유형** | 미국식 영어와 영국식 영어 IPA 및 발음 선택 |
| **문장 읽기 속도** | 문장 읽기 속도 조정(단어 읽기는 영향 없음) |
| **약형/강형 표시** | 기능어의 약형 및 강형 발음 변형 표시 |
| **호버 툴팁** | 마우스 호버 시 IPA 툴팁 표시 |

## 자주 묻는 질문

**Q: 일부 페이지에서 작동하지 않는 이유는?**  
A: 보안상의 이유로 브라우저 확장 프로그램은 `chrome://`, 브라우저 설정, Chrome Web Store 등 특수 페이지에서 실행할 수 없습니다.

**Q: 일부 단어에 IPA가 없는 경우는?**  
A: IPA 사전은 일반적인 영어 단어를 다룹니다. 사전에 없는 단어의 경우, 확장 프로그램이 레마타이제이션 및 G2P를 통해 근사 IPA를 생성하며, 툴팁에 ≈ 또는 ~로 표시됩니다.

**Q: 텍스트 음성 변환에 소리가 없어요?**  
A: 시스템 볼륨 설정을 확인하고 영어 음성 팩이 설치되어 있는지 확인해 주세요. 브라우저와 운영 체제에 따라 음성 지원이 다릅니다.

**Q: 전체 페이지 모드가 레이아웃에 영향을 주나요?**  
A: IPA 주석에는 추가 공간이 필요하며, 원본 페이지 레이아웃에 영향을 줄 수 있습니다. 읽기에 불편하면 전체 페이지 모드를 비활성화하고 호버 툴팁을 대신 사용하세요.


**Q: 툴팁의 ~ 및 ≈ 기호는 무엇을 의미하나요?**  
A: ~는 IPA가 규칙 기반 변환(G2P)으로 생성되었음을, ≈는 관련 기본 단어에서 파생되었음을 나타냅니다. 사전 항목보다 정확도가 낮을 수 있습니다.
