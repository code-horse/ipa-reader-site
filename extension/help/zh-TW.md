# IPA Reader - 使用指南

> 版本：v1.1.0

## 簡介

IPA Reader 是一款專為英語學習者設計的瀏覽器擴充功能。它可以在網頁上的英文單字旁新增 IPA（國際音標）發音註解，支援美式和英式英語口音，幫助您更輕鬆地學習英語發音。

---

## 主要功能

- **選取文字標註** — 在網頁上選取英文文字，自動顯示 IPA 與發音按鈕
- **整頁 IPA 模式** — 一鍵為頁面所有英文單字新增 IPA 註解
- **美式與英式口音** — 可切換美式英語（en-US）和英式英語（en-GB）IPA
- **文字轉語音** — 點擊揚聲器按鈕收聽與所選口音相符的發音
- **選區朗讀** — 選取任意英文文字，點擊浮動按鈕或右鍵選擇「朗讀選取範圍」即可播放
- **懸停提示** — 將滑鼠懸停在已標註單字上可查看 IPA 與發音按鈕
- **多語系介面** — 支援 38 種介面語言

## 使用方式

### 步驟 1：安裝擴充功能

從 [Chrome Web Store](https://chromewebstore.google.com/) 安裝 **IPA Reader**，或在開發者模式下本地載入。

### 步驟 2：開啟任意網頁

造訪包含英文內容的任意網頁。

### 步驟 3：選取文字或使用浮動按鈕

選取需要標註的英文文字，或點擊右下角的浮動按鈕啟用整頁 IPA 模式。

### 步驟 4：查看 IPA

將滑鼠懸停在單字上可查看 IPA 提示框，點擊揚聲器圖示即可收聽發音。

### 步驟 5：朗讀選取文字

用滑鼠選取任意英文文字，點擊出現的🔊浮動按鈕即可朗讀；或右鍵選擇「朗讀選取範圍」。

> **提示：** 點擊瀏覽器工具列中的擴充功能圖示可開啟設定面板，調整口音類型、語速等。

## 選區朗讀

選區朗讀功能讓您選取任意英文文字後一鍵朗讀，非常適合學習句子發音與閱讀練習。

**方法 1：浮動按鈕**  
用滑鼠選取英文文字後，選區右上角會出現一個綠色的🔊按鈕，點擊即可朗讀。

**方法 2：右鍵選單**  
選取英文文字後，右鍵選擇選單中的「朗讀選取範圍」。

> **提示：** 選區朗讀使用您在設定中配置的句子語速。單字發音使用預設語速。TTS 語音與您選擇的口音類型（美式或英式）一致。

## 設定說明

| 設定 | 說明 |
|------|------|
| **啟用 IPA** | 總開關，用於啟用或停用 IPA 標註功能 |
| **整頁 IPA** | 啟用後，為頁面所有英文單字顯示 IPA（可能影響頁面版面配置） |
| **口音類型** | 選擇美式英語或英式英語 IPA 及發音 |
| **句子語速** | 調整句子朗讀速度（單字發音不受影響） |
| **懸停提示** | 滑鼠懸停時顯示 IPA 提示框 |

## 常見問題

**問：為什麼在某些頁面上無法使用？**  
答：基於安全考量，瀏覽器擴充功能無法在特殊頁面執行，例如 `chrome://`、瀏覽器設定頁或 Chrome Web Store。

**問：某些單字沒有 IPA 怎麼辦？**  
答：IPA 詞典涵蓋常用英文單字。部分罕見或專業詞彙可能沒有 IPA 條目。我們會持續擴充詞典。

**問：文字轉語音沒有聲音？**  
答：請檢查系統音量設定，並確保已安裝英語語音套件。不同瀏覽器與作業系統對語音支援各不相同。

**問：整頁模式影響頁面版面配置？**  
答：IPA 註解需要額外空間，可能影響原始頁面版面配置。若影響閱讀，可關閉整頁模式，改用懸停提示。
