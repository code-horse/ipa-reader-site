# IPA Reader - 使用指南

> 版本：v1.1.0

## 簡介

IPA Reader 是一款專為英語學習者設計的瀏覽器擴充功能。它可以在網頁上的英文單字旁新增 IPA（國際音標）發音註解，支援美式和英式英語口音，幫助您更輕鬆地學習英語發音。

---

## 主要功能

- **全網頁音標模式** — 一鍵為頁面所有英文單字新增音標註解，音標符號按類型著色顯示（母音、子音、重音符號），清晰易讀
- **強弱讀效果** — 自動顯示常見虛詞的強讀和弱讀發音變體（如 the、to、can 等），幫助掌握自然連讀發音
- **美式與英式口音** — 可切換美式英語（en-US）和英式英語（en-GB）音標
- **文字轉語音** — 點擊揚聲器按鈕收聽與所選口音相符的發音
- **選區朗讀** — 選取任意英文文字，點擊浮動按鈕或右鍵選擇「朗讀選取範圍」即可播放
- **懸停提示** — 將滑鼠懸停在單字上可查看帶語義著色的音標與發音按鈕
- **同形異音詞消歧** — 自動辨識多音詞（如 read、live 等），根據上下文選擇正確的讀音
- **多語系介面** — 支援 38 種介面語言

## 使用方式

### 步驟 1：安裝擴充功能

從 [Chrome Web Store](https://chromewebstore.google.com/) 安裝 **IPA Reader**，或在開發者模式下本地載入。

### 步驟 2：開啟任意網頁

造訪包含英文內容的任意網頁。

### 步驟 3：啟用音標

點擊瀏覽器工具列中的擴充功能圖示，開啟「啟用音標」開關，再開啟「全網頁音標」即可為頁面所有單字標註。也可以點擊頁面右下角的懸浮按鈕快速切換。

### 步驟 4：查看音標

將滑鼠懸停在單字上，即可看到帶語義著色的音標提示框，點擊揚聲器圖示收聽發音。對於有強弱讀變體的虛詞，提示框會同時顯示兩種讀音。

### 步驟 5：朗讀選取文字

用滑鼠選取任意英文文字，點擊出現的🔊浮動按鈕即可朗讀；或右鍵選擇「朗讀選取範圍」。

> **提示：** 點擊瀏覽器工具列中的擴充功能圖示可開啟設定面板，調整口音類型、語速等。

---

## 全網頁音標模式

開啟全網頁音標模式後，頁面上的每個英文單字上方都會以 Ruby 注音的形式顯示音標。音標符號按語義著色，方便閱讀：

- **母音** — 藍色高亮顯示
- **子音** — 灰色顯示
- **重音符號**（ˈ ˌ）— 紅色高亮顯示
- **長音符號**（ː）— 紫色高亮顯示

擴充功能會自動調整行高，防止注音與文字重疊，並根據單字長度縮放音標字體大小，保持頁面整潔。

---

## 強弱讀效果

英語中許多常見虛詞有兩種發音方式：

- **弱讀** — 在自然連讀中使用的弱化發音（如 the → /ðə/）
- **強讀** — 在強調或單獨使用時的完整發音（如 the → /ðiː/）

開啟「顯示強弱讀效果」後，將滑鼠懸停在這些單字上，提示框會同時顯示弱讀和強讀兩種發音，分別標註為 WEAK 和 STRONG。這有助於理解自然語流中發音的變化。

涵蓋的單字包括：冠詞（the、a、an）、介詞（to、for、of、from、at、as、than）、連詞（and、or、but）、代詞（you、your、her、him、his、them、us、our、there）、助動詞（am、is、are、was、were、been、can、could、would、should、must、shall、will、do、does、have、has、had）等。

## 選區朗讀

選區朗讀功能讓您選取任意英文文字後一鍵朗讀，非常適合學習句子發音與閱讀練習。

**方法 1：浮動按鈕**  
用滑鼠選取英文文字後，選區右上角會出現一個綠色的🔊按鈕，點擊即可朗讀。

**方法 2：右鍵選單**  
選取英文文字後，右鍵選擇選單中的「朗讀選取範圍」。

> **提示：** 選區朗讀使用您在設定中配置的句子語速。單字發音使用預設語速。TTS 語音與您選擇的口音類型（美式或英式）一致。

## 設定說明

| 設定 | 說明 |
|------|------|
| **啟用音標** | 總開關，用於啟用或停用音標標註功能 |
| **全網頁音標** | 啟用後，為頁面所有英文單字顯示帶語義著色的音標 |
| **音標樣式** | 選擇美式英語或英式英語音標及發音 |
| **顯示懸停提示框** | 滑鼠懸停時顯示音標提示框與發音按鈕 |
| **顯示強弱讀效果** | 顯示常見虛詞的強讀和弱讀發音變體 |
| **句子朗讀語速** | 調整句子朗讀速度（單字發音不受影響） |

## 常見問題

**問：為什麼在某些頁面上無法使用？**  
答：基於安全考量，瀏覽器擴充功能無法在特殊頁面執行，例如 `chrome://`、瀏覽器設定頁或 Chrome Web Store。

**問：某些單字沒有音標怎麼辦？**  
答：音標詞典涵蓋常用英文單字。對於詞典中沒有的單字，擴充功能會透過詞形還原和規則式轉換（G2P）產生近似音標，在提示框中以 ≈ 或 ~ 標記。

**問：文字轉語音沒有聲音？**  
答：請檢查系統音量設定，並確保已安裝英語語音套件。不同瀏覽器與作業系統對語音支援各不相同。

**問：全網頁音標模式影響頁面版面配置？**  
答：音標註解需要額外空間。擴充功能會自動調整行高以減小版面影響。若仍影響閱讀，可關閉全網頁音標模式，改用懸停提示。

**問：提示框中的 ~ 和 ≈ 符號是什麼意思？**  
答：~ 表示該音標由規則式轉換（G2P）產生，而非來自詞典。≈ 表示該音標透過詞形還原從相關基礎詞推導而來。這些音標的準確性可能略低於詞典條目。
