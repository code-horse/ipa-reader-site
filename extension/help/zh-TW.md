# IPA Reader - 使用指南

> 版本：v1.4.1

## 簡介

IPA Reader 是一款專為英語學習者設計的瀏覽器擴充功能。它可以在網頁和 PDF 上的英文單字旁新增 IPA（國際音標）發音註解，支援美式和英式英語口音。同時內建英文詞典、文字轉語音和翻譯功能，幫助您更輕鬆地學習英語發音。

---

## 主要功能

- **全網頁音標模式** — 一鍵為頁面所有英文單字新增音標註解，音標符號按類型著色顯示（母音、子音、重音符號），清晰易讀
- **懸浮詞典** — 滑鼠懸停即可查看英文釋義（WordNet 82,000+ 詞條）、語義著色音標和發音按鈕；可選擇詞典模式、提示框模式或關閉
- **PDF 閱讀器** — 內建 PDF 閱讀器，支援音標標註、詞典、語音朗讀和翻譯；支援拖曳開啟、URL 載入和 PDF 智慧偵測自動跳轉
- **強弱讀效果** — 自動顯示常見虛詞的強讀和弱讀發音變體（如 the、to、can 等），幫助掌握自然連讀發音
- **美式與英式口音** — 可切換美式英語（en-US）和英式英語（en-GB）音標
- **文字轉語音** — 點擊揚聲器按鈕收聽與所選口音相符的發音
- **選區朗讀與卡拉 OK 效果** — 選取任意英文文字後會出現精簡工具列，含朗讀與翻譯按鈕；朗讀時會隨音訊即時逐字高亮（卡拉 OK 效果）
- **選區翻譯** — 選取任意文字後，點工具列上的翻譯按鈕，即可透過 Bing 或 Google 翻譯取得即時譯文，以內嵌氣泡顯示
- **同形異音詞消歧** — 自動辨識多音詞（如 read、live），依上下文選擇正確讀音
- **鍵盤快捷鍵** — 可自訂快捷鍵，快速使用核心功能
- **多語系介面** — 支援 41 種介面語言

## 使用方式

### 步驟 1：安裝擴充功能

從 [Chrome Web Store](https://chromewebstore.google.com/) 安裝 **IPA Reader**，或在開發者模式下本地載入。

### 步驟 2：開啟任意網頁

造訪包含英文內容的任意網頁。

### 步驟 3：啟用音標

點擊瀏覽器工具列中的擴充功能圖示。開啟「啟用音標」，再開啟「全網頁音標」即可為頁面所有單字標註。您也可以點擊右下角的懸浮按鈕。

### 步驟 4：查看音標

將滑鼠懸停在單字上，即可看到帶語義著色的音標提示框，點擊揚聲器圖示收聽發音。對於有強弱讀變體的虛詞，提示框會同時顯示兩種讀音。

### 步驟 5：朗讀與翻譯選取文字

用滑鼠選取任意英文文字。選區附近會出現精簡工具列，有兩個按鈕：

- **🔊 朗讀** — 朗讀選取文字，並以卡拉 OK 方式逐字高亮
- **🌐 翻譯** — 在工具列下方顯示內嵌翻譯氣泡

您亦可右鍵選擇「IPA Reader > 朗讀選取範圍」或「IPA Reader > 翻譯選取範圍」。

> **提示：** 點擊瀏覽器工具列中的擴充功能圖示可開啟設定面板，調整口音類型、語速、懸停模式、翻譯引擎等。

## 全網頁音標模式

開啟全網頁音標模式後，頁面上每個英文單字上方都會以 Ruby 注音形式顯示音標。音標符號按語義著色，方便閱讀：

- **母音** — 藍色高亮
- **子音** — 灰色顯示
- **重音符號**（ˈ ˌ）— 紅色高亮
- **長音符號**（ː）— 紫色高亮

擴充功能會自動調整行高，避免注音與正文重疊，並依單字長度縮放音標字體，維持版面整潔。

---

## 懸浮詞典

擴充功能內建基於 WordNet 的英文詞典（82,000+ 詞條）。您可以在設定中選擇不同的懸停模式：

| 模式 | 行為 |
|------|------|
| **詞典模式** | 懸停顯示音標 + 英文釋義 + 發音按鈕 |
| **提示框模式** | 懸停顯示音標 + 發音按鈕（不顯示釋義） |
| **關閉** | 無懸停效果 |

在**詞典模式**下，提示框顯示：
- 單字及其音標
- 發音按鈕（點擊收聽）
- 來自 WordNet 的英文釋義

> **提示：** 詞典資料在啟用詞典模式時按需載入，切換到其他模式時自動卸載以節省記憶體。

## PDF 閱讀器

IPA Reader 內建 PDF 閱讀器，讓您在閱讀 PDF 文件時也能享受音標標註、詞典查詞、語音朗讀和翻譯等功能。

### 開啟 PDF

**方法 1：從彈出面板開啟**  
點擊擴充功能圖示，然後點擊「開啟 PDF 閱讀器」。將 PDF 檔案拖曳至閱讀器中，或點擊「選擇檔案」開啟本地 PDF，也可以直接貼上 PDF 網址。

**方法 2：右鍵選單**  
在網頁上右鍵點擊任意 `.pdf` 連結，選擇「用 IPA Reader 開啟 PDF」。

**方法 3：自動偵測**  
在設定中啟用「PDF 智慧偵測」後，擴充功能會自動將 `.pdf` 連結跳轉到內建閱讀器。對於未自動跳轉的 PDF（如 Chrome 內建檢視器），您會看到通知和提示引導您使用 IPA Reader 開啟。

### PDF 閱讀器功能

- **音標標註** — 所有音標功能均可在 PDF 文字上使用，包括全頁模式和懸停提示
- **點擊查詞** — 點擊任意單字查看釋義（在 PDF 中使用點擊而非懸停，避免閱讀時誤觸）
- **選區工具列** — 選取文字後可朗讀、翻譯或複製
- **側邊欄** — 目錄大綱和頁面縮圖
- **搜尋** — 在 PDF 中搜尋文字
- **主題** — 暗色、淺色和護眼三種閱讀主題
- **縮放** — 多種縮放等級，包括自動適配、適合頁面和適合寬度
- **快捷鍵** — 方向鍵翻頁、+/- 縮放、Ctrl/Cmd+F 搜尋

## 強弱讀效果

英語中許多常見虛詞有兩種發音方式：

- **弱讀** — 在自然連讀中使用的弱化發音（例如 the → /ðə/）
- **強讀** — 在強調或單獨使用時的完整發音（例如 the → /ðiː/）

開啟「顯示強弱讀效果」後，將滑鼠懸停在這些單字上，提示框會同時顯示弱讀與強讀，並標示為「WEAK」與「STRONG」，有助理解自然語流中的發音變化。

涵蓋範圍包括：冠詞（the、a、an）、介詞（to、for、of、from、at、as、than）、連詞（and、or、but）、代詞（you、your、her、him、his、them、us、our、there）、助動詞（am、is、are、was、were、been、can、could、would、should、must、shall、will、do、does、have、has、had）等。

## 選區朗讀與卡拉 OK

選區朗讀可讓您選取任意英文文字並一鍵朗讀，適合學習句子發音與閱讀練習。

**方法 1：選區工具列**  
用滑鼠選取英文文字。選區附近會出現精簡工具列，含 🔊 朗讀與 🌐 翻譯按鈕。點朗讀即可播放。朗讀時每個字會即時高亮（卡拉 OK 效果），方便跟讀。

**方法 2：右鍵選單**  
選取英文文字後，右鍵選擇「IPA Reader > 朗讀選取範圍」。

**方法 3：鍵盤快捷鍵**  
選取文字後按 `Alt+Shift+S`（Mac：`Ctrl+Shift+S`）即可朗讀。

> **提示：** 卡拉 OK 高亮在瀏覽器支援 TTS 字詞邊界事件時效果最佳。若不支援，擴充功能會以計時後備方式維持流暢高亮。

## 翻譯

選取頁面上任意文字即可使用翻譯功能取得即時譯文。

**方法 1：選區工具列**  
選取文字後，點工具列上的 🌐 翻譯按鈕。下方會出現翻譯氣泡顯示結果，並含複製按鈕。

**方法 2：右鍵選單**  
選取文字後，右鍵選擇「IPA Reader > 翻譯選取範圍」。

**方法 3：鍵盤快捷鍵**  
選取文字後按 `Alt+Shift+T`（Mac：`Ctrl+Shift+T`）即可翻譯。

**翻譯引擎：**

- **Bing 翻譯**（預設）— 由 Microsoft Translator 提供
- **Google 翻譯** — 由 Google 提供

兩種引擎均支援 **108 種目標語言**。

您可在擴充功能設定中切換翻譯引擎與目標語言。目標語言會依瀏覽器語言自動偵測。

> **提示：** 在工具列或氣泡外任意處點一下即可關閉。

## 設定說明

| 設定 | 說明 |
|------|------|
| **啟用音標** | 總開關，用於啟用或停用音標標註功能 |
| **全網頁音標** | 啟用後，為頁面所有英文單字在文字上方顯示語義著色音標 |
| **音標樣式** | 選擇美式或英式英語音標與發音 |
| **懸停模式** | 選擇懸停行為：詞典模式（音標+釋義+發音）、提示框模式（音標+發音）或關閉 |
| **顯示強弱讀效果** | 顯示常見虛詞的弱讀與強讀變體 |
| **句子朗讀語速** | 調整句子朗讀速度（單字發音不受影響） |
| **翻譯引擎** | 選擇 Bing 翻譯或 Google 翻譯 |
| **目標語言** | 設定翻譯目標語言（依瀏覽器語言自動偵測） |
| **PDF 智慧偵測** | 啟用後，自動將 PDF 連結跳轉到內建閱讀器，並在偵測到 PDF 時顯示通知 |

## 常見問題

**問：為什麼在某些頁面上無法使用？**  
答：基於安全考量，瀏覽器擴充功能無法在特殊頁面執行，例如 `chrome://`、瀏覽器設定頁或 Chrome Web Store。

**問：某些單字沒有音標怎麼辦？**  
答：音標詞典涵蓋 134,000+ 美式和 67,000+ 英式英語單字。對於詞典中沒有的單字，擴充功能會透過詞形還原與規則式 G2P 轉換產生近似音標，在提示框中以 ≈ 或 ~ 標記。

**問：文字轉語音沒有聲音？**  
答：請檢查系統音量設定，並確保已安裝英語語音套件。不同瀏覽器與作業系統對語音支援各不相同。

**問：全網頁音標模式影響頁面版面？**  
答：音標註解需要額外空間。擴充功能會自動調整行高以減輕版面影響。若仍影響閱讀，可關閉全網頁音標模式，改用懸停提示。

**問：提示框中的 ~ 和 ≈ 符號是什麼意思？**  
答：~ 表示該音標由規則式轉換（G2P）產生，而非來自詞典。≈ 表示該音標透過詞形還原從相關基礎詞推導而來。這些可能比詞典條目略不精確。

**問：翻譯無法使用？**  
答：翻譯需要網路連線。若 Bing 翻譯失敗，請在設定中改試 Google 翻譯。部分網路環境可能無法連線至翻譯服務。

**問：如何用 IPA Reader 開啟 PDF？**  
答：有多種方式：在彈出面板中點擊「開啟 PDF 閱讀器」、右鍵點擊 PDF 連結選擇「用 IPA Reader 開啟 PDF」，或在設定中啟用「PDF 智慧偵測」自動將 PDF 連結跳轉到內建閱讀器。

**問：啟用了 PDF 智慧偵測但某些 PDF 沒有跳轉？**  
答：自動跳轉適用於以 `.pdf` 結尾的連結。對於沒有 `.pdf` 副檔名的 PDF 或透過 Chrome 內建檢視器檢視的 PDF，您會看到通知和徽章提示，引導您在 IPA Reader 中開啟。

**問：詞典可以離線使用嗎？**  
答：可以。WordNet 詞典（82,000+ 詞條）完全內建於擴充功能中，所有查詢均在本地完成，無需網路連線。
