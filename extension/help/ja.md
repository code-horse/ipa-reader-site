# IPA Reader - ユーザーガイド

> バージョン：v1.1.0

## はじめに

IPA Reader は英語学習者向けに設計されたブラウザ拡張機能です。ウェブページ上の英単語に IPA（国際音声記号）の発音注釈を追加し、アメリカ英語とイギリス英語の両方に対応。英語の発音学習をより簡単にします。

---

## 主な機能

- **テキスト選択注釈** — ウェブページで英語テキストを選択すると、IPA と音声ボタンが自動表示
- **全ページ IPA モード** — クリック一つでページ内の全英単語に IPA 注釈を追加
- **アメリカ・イギリス発音** — アメリカ英語（en-US）とイギリス英語（en-GB）の IPA を切り替え可能
- **テキスト読み上げ** — スピーカーボタンをクリックして選択した発音で聞く
- **選択範囲読み上げ** — 任意の英語テキストを選択し、フローティングボタンまたは右クリック「選択範囲を読み上げ」で再生
- **ホバーツールチップ** — 注釈付き単語にマウスを乗せると IPA と音声ボタンが表示
- **多言語対応** — 38 言語のインターフェースをサポート

## 使い方

### ステップ 1：拡張機能をインストール

[Chrome Web Store](https://chromewebstore.google.com/) から **IPA Reader** をインストールするか、開発者モードでローカル読み込み。

### ステップ 2：任意のウェブページを開く

英語コンテンツを含む任意のウェブページにアクセスします。

### ステップ 3：テキストを選択するかフローティングボタンを使用

注釈したい英語テキストを選択するか、右下のフローティングボタンをクリックして全ページ IPA モードを有効化します。

### ステップ 4：IPA を表示

単語にマウスを乗せると IPA ツールチップが表示され、スピーカーアイコンをクリックすると発音を聞けます。

### ステップ 5：選択したテキストを読み上げ

マウスで任意の英語テキストを選択し、フローティングスピーカーボタンをクリックして読み上げ。または右クリックして「選択範囲を読み上げ」を選択。

> **ヒント：** ブラウザのツールバーにある拡張機能アイコンをクリックすると設定パネルが開き、発音タイプや読み上げ速度などを調整できます。

## 選択範囲読み上げ

選択範囲読み上げ機能では、任意の英語テキストを選択してワンクリックで読み上げ可能。文の発音学習やリーディング練習に最適です。

**方法 1：フローティングボタン**  
マウスで英語テキストを選択すると、選択範囲の右上にスピーカーボタンが表示されます — クリックで読み上げ。

**方法 2：右クリックメニュー**  
英語テキストを選択した後、右クリックしてメニューから「選択範囲を読み上げ」を選択。

> **ヒント：** 選択範囲読み上げは設定で構成した文の読み上げ速度を使用します。単語発音はデフォルト速度を使用。TTS 音声は選択した発音タイプ（アメリカまたはイギリス）に合わせます。

## 設定ガイド

| 設定 | 説明 |
|------|------|
| **IPA を有効化** | IPA 注釈機能のオン／オフを切り替えるマスタースイッチ |
| **全ページ IPA** | 有効にすると、ページ内の全英単語に IPA を表示（ページレイアウトに影響する場合あり） |
| **発音タイプ** | アメリカ英語とイギリス英語の IPA および発音を選択 |
| **文の読み上げ速度** | 文の読み上げ速度を調整（単語の発音には影響しない） |
| **ホバーツールチップ** | マウスホバーで IPA ツールチップを表示 |

## よくある質問

**Q：一部のページで動作しないのはなぜ？**  
A：セキュリティ上の理由により、ブラウザ拡張機能は `chrome://`、ブラウザ設定、Chrome Web Store などの特殊ページでは実行できません。

**Q：一部の単語に IPA が表示されない場合は？**  
A：IPA 辞書は一般的な英単語をカバーしています。まれな単語や専門用語は IPA エントリがない場合があります。辞書の拡充を継続しています。

**Q：テキスト読み上げで音が出ない？**  
A：システムの音量設定を確認し、英語の音声パックがインストールされているか確認してください。ブラウザやオペレーティングシステムによって音声サポートは異なります。

**Q：全ページモードでレイアウトが崩れる？**  
A：IPA 注釈には余分なスペースが必要なため、元のページレイアウトに影響する場合があります。読みにくい場合は全ページモードを無効にして、ホバーツールチップを代わりに使用してください。
