# IPA Reader - ユーザーガイド

> バージョン：v1.3.0

## はじめに

IPA Reader は英語学習者向けに設計されたブラウザ拡張機能です。ウェブページ上の英単語に IPA（国際音声記号）の発音注釈を追加し、アメリカ英語とイギリス英語の両方に対応。英語の発音学習をより簡単にします。

---

## 主な機能

- **全ページ IPA モード** — クリック一つでページ内の全英単語に IPA 注釈を追加、IPA シンボルを種類別に色分け表示（母音、子音、アクセント記号）で読みやすく
- **弱形/強形表示** — 機能語（the、to、can など）の弱形・強形の発音バリエーションを自動表示し、自然なつながった話し方の習得をサポート
- **アメリカ・イギリス発音** — アメリカ英語（en-US）とイギリス英語（en-GB）の IPA を切り替え可能
- **テキスト読み上げ** — スピーカーボタンをクリックして選択した発音で聞く
- **カラオケ付き選択範囲読み上げ** — 英語テキストを選択するとコンパクトなツールバーが表示され、読み上げと翻訳ボタンがあります。読み上げ中は音声に同期したリアルタイムの単語ごとのハイライト（カラオケ効果）が表示されます
- **選択範囲の翻訳** — 任意のテキストを選択し、ツールバーの翻訳ボタンをクリックすると、Bing または Google 翻訳による即時翻訳がインラインのバブルに表示されます
- **ホバーツールチップ** — 注釈付き単語にマウスを乗せると IPA と音声ボタンが表示
- **同綴異音語の識別** — 複数の読み方を持つ単語（read、live など）を自動識別し、文脈に基づいて正しい発音を選択
- **キーボードショートカット** — カスタマイズ可能なショートカットで主要機能にすばやくアクセス
- **多言語対応** — 38 言語のインターフェースをサポート

## 使い方

### ステップ 1：拡張機能をインストール

[Chrome Web Store](https://chromewebstore.google.com/) から **IPA Reader** をインストールするか、開発者モードでローカル読み込み。

### ステップ 2：任意のウェブページを開く

英語コンテンツを含む任意のウェブページにアクセスします。

### ステップ 3：IPA を有効にする

ブラウザのツールバーにある拡張機能アイコンをクリックします。「IPA を有効化」をオンにし、「全ページ IPA」をオンにするとページ内のすべての単語に注釈が付きます。右下のフローティングボタンからも切り替えできます。

### ステップ 4：IPA を表示

単語にマウスを乗せると IPA ツールチップが表示され、スピーカーアイコンをクリックすると発音を聞けます。弱形/強形がある機能語では、ツールチップに両方のバリエーションが表示されます。

### ステップ 5：選択したテキストを読み上げ・翻訳

マウスで任意の英語テキストを選択します。選択付近にコンパクトなツールバーが表示され、次の 2 つのボタンがあります。
- **🔊 読み上げ** — 選択テキストを読み上げ、カラオケ風に単語ごとにハイライトします
- **🌐 翻訳** — ツールバーの下にインラインの翻訳バブルを表示します

右クリックから「IPA Reader > Speak Selection」または「IPA Reader > Translate Selection」を選ぶこともできます。

> **ヒント：** ブラウザのツールバーにある拡張機能アイコンをクリックすると設定パネルが開き、アクセント種類、読み上げ速度、翻訳エンジンなどを調整できます。

## 全ページ IPA モード

全ページ IPA モードを有効にすると、ページ上の各英単語の上にルビ形式で IPA が表示されます。IPA シンボルは種類別に色分けされています。

- **母音** — 青で強調
- **子音** — グレー表示
- **アクセント記号**（ˈ ˌ）— 赤で強調
- **長音記号**（ː）— 紫で強調

拡張機能は行間を自動調整し、注釈が本文と重ならないようにし、単語の長さに応じて IPA のフォントサイズを調整してレイアウトを整えます。

---

## 弱形/強形表示

英語の多くの機能語には 2 通りの発音があります。

- **弱形** — 自然なつながった発話で使われる弱化した発音（例：the → /ðə/）
- **強形** — 強調や単独で使うときの完全な発音（例：the → /ðiː/）

「弱形/強形を表示」が有効なとき、これらの単語にマウスを乗せるとツールチップに「WEAK」と「STRONG」のラベル付きで両方が表示されます。

対象には冠詞（the、a、an）、前置詞（to、for、of、from、at、as、than）、接続詞（and、or、but）、代名詞（you、your、her、him、his、them、us、our、there）、助動詞（am、is、are、was、were、been、can、could、would、should、must、shall、will、do、does、have、has、had）などが含まれます。

---

## 選択範囲読み上げとカラオケ

選択範囲の読み上げでは、英語テキストを選んでワンクリックで読み上げできます。文の発音学習やリーディング練習に適しています。

**方法 1：選択ツールバー**  
マウスで英語テキストを選択します。選択付近にコンパクトなツールバーが表示され、🔊 読み上げと 🌐 翻訳のボタンがあります。読み上げをクリックして再生します。読み上げ中は各単語がリアルタイムでハイライトされます（カラオケ効果）。

**方法 2：右クリックメニュー**  
英語テキストを選択した後、右クリックして「IPA Reader > Speak Selection」を選択します。

**方法 3：キーボードショートカット**  
テキストを選択して `Alt+Shift+S`（Mac：`Ctrl+Shift+S`）を押します。

> **ヒント：** カラオケのハイライトは、ブラウザが TTS の単語境界イベントをサポートしているときに最も正確です。未対応の場合は、タイミングに基づくフォールバックでスムーズにハイライトします。

---

## 翻訳

ページ上の任意のテキストを選択して、翻訳機能で即時に訳文を得られます。

**方法 1：選択ツールバー**  
テキストを選択し、ツールバーの 🌐 翻訳をクリックします。結果がツールバーの下にバブルで表示され、コピーボタンがあります。

**方法 2：右クリックメニュー**  
テキストを選択し、右クリックして「IPA Reader > Translate Selection」を選びます。

**方法 3：キーボードショートカット**  
テキストを選択して `Alt+Shift+T`（Mac：`Ctrl+Shift+T`）を押します。

**翻訳エンジン：**
- **Bing 翻訳**（既定）— Microsoft Translator による提供
- **Google 翻訳** — Google による提供

両エンジンとも **108 の翻訳先言語** に対応しています。

拡張機能の設定で翻訳エンジンと目標言語を切り替えられます。目標言語はブラウザの言語から自動検出されます。

> **ヒント：** ツールバーやバブルの外側をクリックすると閉じます。

## キーボードショートカット

| ショートカット | Mac のショートカット | 操作 |
|----------|-------------|--------|
| `Alt+Shift+I` | `Ctrl+Shift+I` | IPA 注釈のオン/オフ |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 選択範囲を読み上げ |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 選択範囲を翻訳 |

> **ヒント：** Chrome の `chrome://extensions/shortcuts` でこれらのショートカットをカスタマイズできます。

## 設定ガイド

| 設定 | 説明 |
|------|------|
| **IPA を有効化** | IPA 注釈機能のオン／オフを切り替えるマスタースイッチ |
| **全ページ IPA** | 有効にすると、ページ内の全英単語に色分けされた IPA を本文の上に表示 |
| **発音タイプ** | アメリカ英語とイギリス英語の IPA および発音を選択 |
| **ホバーツールチップ** | マウスホバーで IPA ツールチップと発音ボタンを表示 |
| **弱形/強形を表示** | 機能語の弱形・強形の発音バリエーションを表示 |
| **文の読み上げ速度** | 文の読み上げ速度を調整（単語の発音には影響しない） |
| **翻訳エンジン** | Bing 翻訳と Google 翻訳から選択 |
| **目標言語** | 翻訳の目標言語を設定（ブラウザの言語から自動検出） |

## よくある質問

**Q：一部のページで動作しないのはなぜ？**  
A：セキュリティ上の理由により、ブラウザ拡張機能は `chrome://`、ブラウザ設定、Chrome Web Store などの特殊ページでは実行できません。

**Q：一部の単語に IPA が表示されない場合は？**  
A: IPA 辞書は一般的な英単語をカバーしています。辞書にない単語については、拡張機能がレンマ化と G2P によって近似的な IPA を生成し、ツールチップに ≈ または ~ で表示されます。

**Q：テキスト読み上げで音が出ない？**  
A：システムの音量設定を確認し、英語の音声パックがインストールされているか確認してください。ブラウザやオペレーティングシステムによって音声サポートは異なります。

**Q：全ページモードでレイアウトが崩れる？**  
A：IPA 注釈には余分なスペースが必要です。拡張機能は行間を調整して影響を抑えます。読みにくい場合は全ページモードを無効にし、ホバーツールチップを使ってください。

**Q: ツールチップの ~ と ≈ の記号は何を意味しますか？**  
A: ~ は IPA がルールベース変換（G2P）で生成されたことを、≈ は関連する基本語から派生されたことを示します。辞書エントリほど正確でない場合があります。

**Q：翻訳が動作しません。**  
A：翻訳にはインターネット接続が必要です。Bing 翻訳が失敗する場合は、設定で Google 翻訳に切り替えてみてください。一部のネットワーク環境では翻訳サービスへのアクセスが遮断されることがあります。
