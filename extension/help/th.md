# IPA Reader - คู่มือผู้ใช้

> เวอร์ชัน: v1.3.0

## คำนำ

IPA Reader เป็นส่วนขยายเบราว์เซอร์ที่ออกแบบมาสำหรับผู้เรียนภาษาอังกฤษ เพิ่มคำอธิบายการออกเสียง IPA (สัทอักษรสากล) ให้กับคำภาษาอังกฤษบนหน้าเว็บ รองรับทั้งสำเนียงอเมริกันและอังกฤษ ช่วยให้คุณเรียนการออกเสียงภาษาอังกฤษได้ง่ายขึ้น

---

## คุณสมบัติหลัก

- **โหมด IPA ทั้งหน้า** — เพิ่มคำอธิบาย IPA ให้คำภาษาอังกฤษทั้งหมดในหน้าด้วยคลิกเดียว สัญลักษณ์ IPA แยกสีตามประเภท (สระ พยัญชนะ เครื่องหมายเน้นเสียง) เพื่อการอ่านง่าย
- **รูปแบบอ่อน/เข้ม** — แสดงตัวแปรการออกเสียงแบบอ่อนและเข้มของคำฟังก์ชัน (เช่น „the”, „to”, „can”) โดยอัตโนมัติ เพื่อฝึกการพูดเชื่อมต่อธรรมชาติ
- **สำเนียงอเมริกันและอังกฤษ** — สลับระหว่าง IPA ภาษาอังกฤษอเมริกัน (en-US) และอังกฤษ (en-GB)
- **แปลงข้อความเป็นเสียง** — คลิกปุ่มลำโพงเพื่อฟังการออกเสียงให้ตรงกับสำเนียงที่คุณเลือก
- **อ่านข้อความที่เลือกพร้อมเอฟเฟกต์คาราโอเกะ** — เลือกข้อความภาษาอังกฤษใดๆ แถบเครื่องมือแบบกะทัดรัดจะปรากฏพร้อมปุ่มอ่านและแปล เสียงจะเล่นพร้อมไฮไลต์ทีละคำแบบเรียลไทม์ (เอฟเฟกต์คาราโอเกะ) ซิงค์กับเสียง
- **แปลข้อความที่เลือก** — เลือกข้อความใดๆ คลิกปุ่มแปลบนแถบเครื่องมือเพื่อแปลทันทีผ่าน Bing หรือ Google Translate แสดงในฟองบับเบิลแบบอินไลน์
- **คำแนะนำเมื่อวางเมาส์** — วางเมาส์เหนือคำที่มีคำอธิบายเพื่อดู IPA พร้อมสัญลักษณ์แยกสีและปุ่มออกเสียง
- **การแยกแยะคำพ้องรูป** — ระบุคำที่มีหลายการออกเสียง (เช่น read, live) โดยอัตโนมัติ และเลือกการออกเสียงที่ถูกต้องตามบริบท
- **แป้นพิมพ์ลัด** — เข้าถึงฟีเจอร์หลักได้อย่างรวดเร็วผ่านแป้นพิมพ์ลัดที่ปรับแต่งได้
- **อินเทอร์เฟซหลายภาษา** — รองรับ 38 ภาษา

## วิธีใช้

### ขั้นตอนที่ 1: ติดตั้งส่วนขยาย

ติดตั้ง **IPA Reader** จาก [Chrome Web Store](https://chromewebstore.google.com/) หรือโหลดแบบท้องถิ่นในโหมดนักพัฒนา

### ขั้นตอนที่ 2: เปิดหน้าเว็บใดก็ได้

เยี่ยมชมหน้าเว็บใดๆ ที่มีเนื้อหาภาษาอังกฤษ

### ขั้นตอนที่ 3: เปิด IPA

คลิกไอคอนส่วนขยายบนแถบเครื่องมือเบราว์เซอร์ เปิด „เปิด IPA” จากนั้นเปิด „IPA ทั้งหน้า” เพื่อใส่คำอธิบายให้ทุกคำบนหน้า คุณยังใช้ปุ่มลอยมุมล่างขวาได้

### ขั้นตอนที่ 4: ดู IPA

วางเมาส์เหนือคำเพื่อดูทูลทิป IPA พร้อมสัญลักษณ์โฟเนติกแยกสี คลิกไอคอนลำโพงเพื่อฟังการออกเสียง สำหรับคำที่มีรูปแบบอ่อน/เข้ม ทูลทิปแสดงทั้งสองแบบ

### ขั้นตอนที่ 5: อ่านและแปลข้อความที่เลือก

เลือกข้อความภาษาอังกฤษด้วยเมาส์ แถบเครื่องมือแบบกะทัดรัดจะปรากฏใกล้การเลือกพร้อมสองปุ่ม:
- **🔊 อ่าน** — อ่านข้อความที่เลือกพร้อมไฮไลต์ทีละคำแบบคาราโอเกะ
- **🌐 แปล** — แสดงฟองแปลแบบอินไลน์ใต้แถบเครื่องมือ

หรือคลิกขวาแล้วเลือก „IPA Reader > Speak Selection” หรือ „IPA Reader > Translate Selection”

> **เคล็ดลับ:** คลิกไอคอนส่วนขยายบนแถบเครื่องมือเพื่อเปิดแผงการตั้งค่า: ประเภทสำเนียง อัตราความเร็วการอ่าน เครื่องมือแปล และอื่นๆ

## โหมด IPA ทั้งหน้า

เมื่อเปิดโหมด IPA ทั้งหน้า คำภาษาอังกฤษทุกคำบนหน้าจะมีคำอธิบาย IPA อยู่เหนือคำแบบ ruby สัญลักษณ์ IPA แยกสีเพื่อการอ่านง่าย:

- **สระ** — เน้นสีน้ำเงิน
- **พยัญชนะ** — สีเทา
- **เครื่องหมายเน้นเสียง** (ˈ ˌ) — เน้นสีแดง
- **เครื่องหมายความยาวเสียง** (ː) — เน้นสีม่วง

ส่วนขยายปรับความสูงบรรทัดอัตโนมัติเพื่อไม่ให้คำอธิบายทับข้อความ และปรับขนาดตัวอักษร IPA ตามความยาวคำ

## รูปแบบอ่อน/เข้ม

คำฟังก์ชันทั่วไปหลายคำมีสองการออกเสียง:

- **รูปแบบอ่อน** — การออกเสียงย่อในคำพูดต่อเนื่อง (เช่น „the” → /ðə/)
- **รูปแบบเข้ม** — การออกเสียงเต็มเมื่อเน้นหรือพูดเดี่ยว (เช่น „the” → /ðiː/)

เมื่อเปิด „แสดงรูปแบบอ่อน/เข้ม” การวางเมาส์เหนือคำเหล่านี้จะแสดงทั้งสองแบบในทูลทิป พร้อมป้าย „WEAK” และ „STRONG” ช่วยให้เข้าใจการเปลี่ยนการออกเสียงในคำพูดจริง

ครอบคลุมเช่น: อนุภาค (the, a, an), บุพบท (to, for, of, from, at, as, than), สันธาน (and, or, but), สรรพนาม (you, your, her, him, his, them, us, our, there), กริยาช่วย (am, is, are, was, were, been, can, could, would, should, must, shall, will, do, does, have, has, had) และอื่นๆ

## การอ่านข้อความที่เลือกและคาราโอเกะ

คุณสมบัติการอ่านข้อความที่เลือกช่วยให้เลือกข้อความภาษาอังกฤษใดๆ และอ่านออกเสียงด้วยคลิกเดียว เหมาะสำหรับเรียนรู้การออกเสียงประโยคและการฝึกอ่าน

**วิธีที่ 1: แถบเครื่องมือเมื่อเลือก**  
เลือกข้อความภาษาอังกฤษด้วยเมาส์ แถบกะทัดรัดพร้อมปุ่ม 🔊 อ่านและปุ่ม 🌐 แปลจะปรากฏใกล้การเลือก คลิกอ่านเพื่อเล่น ขณะอ่านแต่ละคำจะถูกไฮไลต์แบบเรียลไทม์ (เอฟเฟกต์คาราโอเกะ)

**วิธีที่ 2: เมนูคลิกขวา**  
หลังเลือกข้อความภาษาอังกฤษ คลิกขวาแล้วเลือก „IPA Reader > Speak Selection”

**วิธีที่ 3: แป้นพิมพ์ลัด**  
เลือกข้อความแล้วกด `Alt+Shift+S` (Mac: `Ctrl+Shift+S`) เพื่ออ่าน

> **เคล็ดลับ:** ไฮไลต์แบบคาราโอเกะทำงานได้ดีที่สุดเมื่อเบราว์เซอร์รองรับเหตุการณ์ขอบเขตคำของ TTS หากไม่รองรับ ส่วนขยายจะใช้การสำรองตามเวลาเพื่อไฮไลต์ให้ลื่นไหล

## การแปล

เลือกข้อความใดๆ บนหน้าแล้วใช้การแปลเพื่อรับผลลัพธ์ทันที

**วิธีที่ 1: แถบเครื่องมือเมื่อเลือก**  
เลือกข้อความ แล้วคลิกปุ่ม 🌐 แปลบนแถบ ฟองแปลจะปรากฏด้านล่างพร้อมปุ่มคัดลอก

**วิธีที่ 2: เมนูคลิกขวา**  
เลือกข้อความ คลิกขวาแล้วเลือก „IPA Reader > Translate Selection”

**วิธีที่ 3: แป้นพิมพ์ลัด**  
เลือกข้อความแล้วกด `Alt+Shift+T` (Mac: `Ctrl+Shift+T`) เพื่อแปล

**เครื่องมือแปล:**
- **Bing Translate** (ค่าเริ่มต้น) — Microsoft Translator
- **Google Translate** — Google

สลับเครื่องมือแปลและภาษาปลายทางได้ในการตั้งค่าส่วนขยาย ภาษาปลายทางตรวจจับอัตโนมัติจากภาษาของเบราว์เซอร์

> **เคล็ดลับ:** คลิกนอกแถบเครื่องมือหรือฟองเพื่อปิด

---

## แป้นพิมพ์ลัด

| แป้นพิมพ์ลัด | Mac | การทำงาน |
|--------------|-----|-----------|
| `Alt+Shift+I` | `Ctrl+Shift+I` | เปิด/ปิดคำอธิบาย IPA |
| `Alt+Shift+S` | `Ctrl+Shift+S` | อ่านข้อความที่เลือก |
| `Alt+Shift+T` | `Ctrl+Shift+T` | แปลข้อความที่เลือก |

> **เคล็ดลับ:** ปรับแต่งแป้นพิมพ์ลัดใน Chrome ที่ `chrome://extensions/shortcuts`

## คู่มือการตั้งค่า

| การตั้งค่า | คำอธิบาย |
|-----------|----------|
| **เปิด IPA** | สวิตช์หลักเพื่อเปิดหรือปิดคุณสมบัติคำอธิบาย IPA |
| **IPA ทั้งหน้า** | เมื่อเปิดจะแสดง IPA แยกสีเหนือคำภาษาอังกฤษทั้งหมด |
| **รูปแบบ IPA** | เลือกระหว่าง IPA และการออกเสียงภาษาอังกฤษอเมริกันและอังกฤษ |
| **แสดงทูลทิปเมื่อวางเมาส์** | ทูลทิป IPA พร้อมปุ่มออกเสียงเมื่อวางเมาส์ |
| **แสดงรูปแบบอ่อน/เข้ม** | ตัวแปรการออกเสียงแบบอ่อนและเข้มของคำฟังก์ชัน |
| **อัตราความเร็วการอ่านประโยค** | ปรับความเร็วการอ่านประโยค (การออกเสียงคำเดียวไม่เปลี่ยน) |
| **เครื่องมือแปล** | เลือกระหว่าง Bing Translate และ Google Translate |
| **ภาษาปลายทาง** | ตั้งภาษาปลายทางของการแปล (ตรวจจับอัตโนมัติจากภาษาเบราว์เซอร์) |

## คำถามที่พบบ่อย

**ถาม: ทำไมไม่ทำงานในบางหน้า?**  
ตอบ: ด้วยเหตุผลด้านความปลอดภัย ส่วนขยายเบราว์เซอร์ไม่สามารถทำงานบนหน้าพิเศษเช่น `chrome://` การตั้งค่าเบราว์เซอร์ หรือ Chrome Web Store ได้

**ถาม: ถ้าไม่มี IPA สำหรับบางคำล่ะ?**  
ตอบ: พจนานุกรม IPA ครอบคลุมคำภาษาอังกฤษทั่วไป สำหรับคำที่ไม่อยู่ในพจนานุกรม ส่วนขยายสร้าง IPA โดยประมาณผ่าน lemmatization และ G2P ตามกฎ โดยมีเครื่องหมาย ≈ หรือ ~ ในทูลทิป

**ถาม: ไม่มีเสียงจากการแปลงข้อความเป็นเสียง?**  
ตอบ: กรุณาตรวจสอบการตั้งค่าปริมาณเสียงของระบบและตรวจสอบว่าติดตั้งแพ็กเกจเสียงภาษาอังกฤษแล้ว การสนับสนุนเสียงแตกต่างกันไปตามเบราว์เซอร์และระบบปฏิบัติการ

**ถาม: โหมดทั้งหน้ามีผลต่อการจัดวาง?**  
ตอบ: คำอธิบาย IPA ต้องใช้พื้นที่เพิ่ม ส่วนขยายปรับความสูงบรรทัดเพื่อลดผลกระทบ หากยังรบกวนการอ่าน ให้ปิดโหมดทั้งหน้าและใช้คำแนะนำเมื่อวางเมาส์แทน

**ถาม: สัญลักษณ์ ~ และ ≈ ในทูลทิปหมายความว่าอะไร?**  
ตอบ: ~ หมายความว่า IPA ถูกสร้างโดยกฎ (G2P) ไม่ใช่จากพจนานุกรม ≈ หมายความว่าได้มาจากคำฐานที่เกี่ยวข้องผ่าน lemmatization อาจมีความแม่นยำน้อยกว่ารายการในพจนานุกรม

**ถาม: การแปลไม่ทำงาน?**  
ตอบ: การแปลต้องใช้อินเทอร์เน็ต หาก Bing Translate ล้มเหลว ลองสลับเป็น Google Translate ในค่าการตั้งค่า บางเครือข่ายอาจบล็อกการเข้าถึงบริการแปล
