# IPA Reader - 使用指南

> 版本：v1.1.0

## 简介

IPA Reader 是一款专为英语学习者设计的浏览器扩展。它可以在网页上的英文单词旁添加 IPA（国际音标）发音注释，支持美式和英式英语口音，帮助您更轻松地学习英语发音。

---

## 主要功能

- **全网页音标模式** — 一键为页面所有英文单词添加音标注释，音标符号按类型着色显示（元音、辅音、重音符号），清晰易读
- **强弱读效果** — 自动显示常见虚词的强读和弱读发音变体（如 the、to、can 等），帮助掌握自然连读发音
- **美式与英式口音** — 可在美式英语（en-US）和英式英语（en-GB）音标之间切换
- **文本转语音** — 点击扬声器按钮收听与所选口音匹配的发音
- **选区朗读** — 选中任意英文文本，点击浮动按钮或右键选择「朗读选区」即可播放
- **悬停提示** — 将鼠标悬停在单词上可查看带语义着色的音标和发音按钮
- **同形异音词消歧** — 自动识别多音词（如 read、live 等），根据上下文选择正确的读音
- **多语言界面** — 支持 38 种界面语言

## 使用方法

### 步骤 1：安装扩展

从 [Chrome Web Store](https://chromewebstore.google.com/) 安装 **IPA Reader**，或在开发者模式下本地加载。

### 步骤 2：打开任意网页

访问包含英文内容的任意网页。

### 步骤 3：启用音标

点击浏览器工具栏中的扩展图标，打开「启用音标」开关，再打开「全网页音标」即可为页面所有单词标注。也可以点击页面右下角的悬浮按钮快速切换。

### 步骤 4：查看音标

将鼠标悬停在单词上，即可看到带语义着色的音标提示框，点击扬声器图标收听发音。对于有强弱读变体的虚词，提示框会同时显示两种读音。

### 步骤 5：朗读选中文本

用鼠标选中任意英文文本，点击出现的🔊浮动按钮即可朗读；或右键选择「朗读选区」。

> **提示：** 点击浏览器工具栏中的扩展图标可打开设置面板，调整口音类型、语速等。

## 全网页音标模式

开启全网页音标模式后，页面上的每个英文单词上方都会以 Ruby 注音的形式显示音标。音标符号按语义着色，方便阅读：

- **元音** — 蓝色高亮显示
- **辅音** — 灰色显示
- **重音符号**（ˈ ˌ）— 红色高亮显示
- **长音符号**（ː）— 紫色高亮显示

扩展会自动调整行高，防止注音与文字重叠，并根据单词长度缩放音标字体大小，保持页面整洁。

## 强弱读效果

英语中许多常见虚词有两种发音方式：

- **弱读** — 在自然连读中使用的弱化发音（如 the → /ðə/）
- **强读** — 在强调或单独使用时的完整发音（如 the → /ðiː/）

开启「显示强弱读效果」后，将鼠标悬停在这些单词上，提示框会同时显示弱读和强读两种发音，分别标注为 WEAK 和 STRONG。这有助于理解自然语流中发音的变化。

涵盖的单词包括：冠词（the、a、an）、介词（to、for、of、from、at、as、than）、连词（and、or、but）、代词（you、your、her、him、his、them、us、our、there）、助动词（am、is、are、was、were、been、can、could、would、should、must、shall、will、do、does、have、has、had）等。

## 选区朗读

选区朗读功能让您选中任意英文文本后一键朗读，非常适合学习句子发音和阅读练习。

**方法 1：浮动按钮**  
用鼠标选中英文文本后，选区右上角会出现一个绿色的🔊按钮，点击即可朗读。

**方法 2：右键菜单**  
选中英文文本后，右键选择菜单中的「朗读选区」。

> **提示：** 选区朗读使用您在设置中配置的句子语速。单词发音使用默认语速。TTS 语音与您选择的口音类型（美式或英式）一致。

## 设置说明

| 设置 | 说明 |
|------|------|
| **启用音标** | 总开关，用于启用或禁用音标标注功能 |
| **全网页音标** | 启用后，为页面所有英文单词显示带语义着色的音标 |
| **音标样式** | 选择美式英语或英式英语音标及发音 |
| **显示悬停提示框** | 鼠标悬停时显示音标提示框和发音按钮 |
| **显示强弱读效果** | 显示常见虚词的强读和弱读发音变体 |
| **句子朗读语速** | 调整句子朗读速度（单词发音不受影响） |

## 常见问题

**问：为什么在某些页面上无法使用？**  
答：出于安全原因，浏览器扩展无法在特殊页面运行，如 `chrome://`、浏览器设置页或 Chrome Web Store。

**问：某些单词没有音标怎么办？**  
答：音标词典涵盖常用英文单词。对于词典中没有的单词，扩展会通过词形还原和规则式转换（G2P）生成近似音标，在提示框中以 ≈ 或 ~ 标记。

**问：文本转语音没有声音？**  
答：请检查系统音量设置，并确保已安装英语语音包。不同浏览器和操作系统对语音支持各不相同。

**问：全网页音标模式影响页面布局？**  
答：音标注释需要额外空间。扩展会自动调整行高以减小布局影响。若仍影响阅读，可关闭全网页音标模式，改用悬停提示。

**问：提示框中的 ~ 和 ≈ 符号是什么意思？**  
答：~ 表示该音标由规则式转换（G2P）生成，而非来自词典。≈ 表示该音标通过词形还原从相关基础词推导而来。这些音标的准确性可能略低于词典条目。
