# IPA Reader - 使用指南

> 版本：v1.4.0

## 简介

IPA Reader 是一款专为英语学习者设计的浏览器扩展。它可以在网页和 PDF 上的英文单词旁添加 IPA（国际音标）发音注释，支持美式和英式英语口音。同时内置英文词典、文本转语音和翻译功能，帮助您更轻松地学习英语发音。

---

## 主要功能

- **全网页音标模式** — 一键为页面所有英文单词添加音标注释，音标符号按类型着色显示（元音、辅音、重音符号），清晰易读
- **悬浮词典** — 鼠标悬停即可查看英文释义（WordNet 82,000+ 词条）、语义着色音标和发音按钮；可选择词典模式、提示框模式或关闭
- **PDF 阅读器** — 内置 PDF 阅读器，支持音标标注、词典、语音朗读和翻译；支持拖拽打开、URL 加载和 PDF 智能检测自动跳转
- **强弱读效果** — 自动显示常见虚词的强读和弱读发音变体（如 the、to、can 等），帮助掌握自然连读发音
- **美式与英式口音** — 可在美式英语（en-US）和英式英语（en-GB）音标之间切换
- **文本转语音** — 点击扬声器按钮收听与所选口音匹配的发音
- **选区朗读 + 卡拉OK效果** — 选中任意英文文本，弹出紧凑工具栏（朗读 + 翻译按钮），朗读时逐词高亮同步显示（卡拉OK效果）
- **选区翻译** — 选中文本后点击工具栏翻译按钮，通过 Bing 或 Google 翻译即时获取翻译结果，以悬浮气泡显示
- **同形异音词消歧** — 自动识别多音词（如 read、live 等），根据上下文选择正确的读音
- **快捷键** — 通过可自定义的键盘快捷键快速使用核心功能
- **多语言界面** — 支持 41 种界面语言

## 使用方法

### 步骤 1：安装扩展

从 [Chrome Web Store](https://chromewebstore.google.com/) 安装 **IPA Reader**，或在开发者模式下本地加载。

### 步骤 2：打开任意网页

访问包含英文内容的任意网页。

### 步骤 3：启用音标

点击浏览器工具栏中的扩展图标，打开「启用音标」开关，再打开「全网页音标」即可为页面所有单词标注。也可以点击页面右下角的悬浮按钮快速切换。

### 步骤 4：查看音标

将鼠标悬停在单词上，即可看到带语义着色的音标提示框，点击扬声器图标收听发音。对于有强弱读变体的虚词，提示框会同时显示两种读音。

### 步骤 5：朗读与翻译选中文本

用鼠标选中任意英文文本，选区附近会出现一个紧凑工具栏，包含两个按钮：
- **🔊 朗读** — 朗读选中文本，同时逐词高亮显示（卡拉OK效果）
- **🌐 翻译** — 在工具栏下方显示翻译结果气泡

也可以右键选择「IPA Reader > 朗读选区」或「IPA Reader > 翻译选区」。

> **提示：** 点击浏览器工具栏中的扩展图标可打开设置面板，调整口音类型、语速、悬停模式、翻译引擎等。

## 全网页音标模式

开启全网页音标模式后，页面上的每个英文单词上方都会以 Ruby 注音的形式显示音标。音标符号按语义着色，方便阅读：

- **元音** — 蓝色高亮显示
- **辅音** — 灰色显示
- **重音符号**（ˈ ˌ）— 红色高亮显示
- **长音符号**（ː）— 紫色高亮显示

扩展会自动调整行高，防止注音与文字重叠，并根据单词长度缩放音标字体大小，保持页面整洁。

## 悬浮词典

扩展内置了基于 WordNet 的英文词典（82,000+ 词条）。您可以在设置中选择不同的悬停模式：

| 模式 | 行为 |
|------|------|
| **词典模式** | 悬停显示音标 + 英文释义 + 发音按钮 |
| **提示框模式** | 悬停显示音标 + 发音按钮（不显示释义） |
| **关闭** | 无悬停效果 |

在**词典模式**下，提示框显示：
- 单词及其音标
- 发音按钮（点击收听）
- 来自 WordNet 的英文释义

> **提示：** 词典数据在启用词典模式时按需加载，切换到其他模式时自动卸载以节省内存。

## PDF 阅读器

IPA Reader 内置 PDF 阅读器，让您在阅读 PDF 文档时也能享受音标标注、词典查词、语音朗读和翻译等功能。

### 打开 PDF

**方法 1：从弹出面板打开**  
点击扩展图标，然后点击「打开 PDF 阅读器」。将 PDF 文件拖拽到阅读器中，或点击「选择文件」打开本地 PDF，也可以直接粘贴 PDF 链接。

**方法 2：右键菜单**  
在网页上右键点击任意 `.pdf` 链接，选择「用 IPA Reader 打开 PDF」。

**方法 3：自动检测**  
在设置中启用「PDF 智能检测」后，扩展会自动将 `.pdf` 链接跳转到内置阅读器。对于未自动跳转的 PDF（如 Chrome 内置查看器），您会看到通知和提示引导您使用 IPA Reader 打开。

### PDF 阅读器功能

- **音标标注** — 所有音标功能均可在 PDF 文本上使用，包括全页模式和悬停提示
- **点击查词** — 点击任意单词查看释义（在 PDF 中使用点击而非悬停，避免阅读时误触）
- **选区工具栏** — 选中文本后可朗读、翻译或复制
- **侧边栏** — 目录大纲和页面缩略图
- **搜索** — 在 PDF 中搜索文本
- **主题** — 暗色、浅色和护眼三种阅读主题
- **缩放** — 多种缩放级别，包括自动适配、适合页面和适合宽度
- **快捷键** — 方向键翻页、+/- 缩放、Ctrl/Cmd+F 搜索

## 强弱读效果

英语中许多常见虚词有两种发音方式：

- **弱读** — 在自然连读中使用的弱化发音（如 the → /ðə/）
- **强读** — 在强调或单独使用时的完整发音（如 the → /ðiː/）

开启「显示强弱读效果」后，将鼠标悬停在这些单词上，提示框会同时显示弱读和强读两种发音，分别标注为 WEAK 和 STRONG。这有助于理解自然语流中发音的变化。

涵盖的单词包括：冠词（the、a、an）、介词（to、for、of、from、at、as、than）、连词（and、or、but）、代词（you、your、her、him、his、them、us、our、there）、助动词（am、is、are、was、were、been、can、could、would、should、must、shall、will、do、does、have、has、had）等。

## 选区朗读与卡拉OK

选区朗读功能让您选中任意英文文本后一键朗读，非常适合学习句子发音和阅读练习。

**方法 1：选区工具栏**  
用鼠标选中英文文本后，选区附近会出现一个紧凑工具栏，包含🔊朗读按钮和🌐翻译按钮。点击朗读按钮播放，播放时每个单词会实时高亮显示（卡拉OK效果），帮助您跟读。

**方法 2：右键菜单**  
选中英文文本后，右键选择「IPA Reader > 朗读选区」。

**方法 3：快捷键**  
选中文本后按 `Alt+Shift+S`（Mac：`Ctrl+Shift+S`）即可朗读。

> **提示：** 卡拉OK高亮效果在浏览器支持 TTS 词边界事件时效果最佳。若不支持，扩展会使用基于计时的备用方案实现平滑高亮。

---

## 翻译

选中页面上的任意文本，使用翻译功能即可获取即时翻译。

**方法 1：选区工具栏**  
选中文本后，点击工具栏中的🌐翻译按钮，下方会弹出翻译结果气泡，带有复制按钮。

**方法 2：右键菜单**  
选中文本后，右键选择「IPA Reader > 翻译选区」。

**方法 3：快捷键**  
选中文本后按 `Alt+Shift+T`（Mac：`Ctrl+Shift+T`）即可翻译。

**翻译引擎：**
- **Bing 翻译**（默认）— 由微软翻译提供支持
- **Google 翻译** — 由 Google 提供支持

两种引擎均支持 **108 种目标语言**。

您可以在扩展设置中切换翻译引擎和目标语言。目标语言会根据您的浏览器语言自动检测。

> **提示：** 点击工具栏或气泡外的任意位置即可关闭。

|------|------|

| **全网页音标** | 启用后，为页面所有英文单词显示带语义着色的音标 |

## 常见问题

**问：为什么在某些页面上无法使用？**  
答：出于安全原因，浏览器扩展无法在特殊页面运行，如 `chrome://`、浏览器设置页或 Chrome Web Store。

**问：某些单词没有音标怎么办？**  
答：音标词典涵盖 134,000+ 美式和 67,000+ 英式英语单词。对于词典中没有的单词，扩展会通过词形还原和规则式转换（G2P）生成近似音标，在提示框中以 ≈ 或 ~ 标记。

**问：文本转语音没有声音？**  
答：请检查系统音量设置，并确保已安装英语语音包。不同浏览器和操作系统对语音支持各不相同。

**问：全网页音标模式影响页面布局？**  
答：音标注释需要额外空间。扩展会自动调整行高以减小布局影响。若仍影响阅读，可关闭全网页音标模式，改用悬停提示。

**问：提示框中的 ~ 和 ≈ 符号是什么意思？**  
答：~ 表示该音标由规则式转换（G2P）生成，而非来自词典。≈ 表示该音标通过词形还原从相关基础词推导而来。这些音标的准确性可能略低于词典条目。

**问：翻译不工作？**  
答：翻译功能需要网络连接。如果 Bing 翻译失败，请尝试在设置中切换为 Google 翻译。某些网络环境可能会阻止访问翻译服务。

**问：如何用 IPA Reader 打开 PDF？**  
答：有多种方式：在弹出面板中点击「打开 PDF 阅读器」、右键点击 PDF 链接选择「用 IPA Reader 打开 PDF」，或在设置中启用「PDF 智能检测」自动将 PDF 链接跳转到内置阅读器。

**问：启用了 PDF 智能检测但某些 PDF 没有跳转？**  
答：自动跳转适用于以 `.pdf` 结尾的链接。对于没有 `.pdf` 扩展名的 PDF 或通过 Chrome 内置查看器查看的 PDF，您会看到通知和徽章提示，引导您在 IPA Reader 中打开。

**问：词典可以离线使用吗？**  
答：可以。WordNet 词典（82,000+ 词条）完全内置于扩展中，所有查询均在本地完成，无需网络连接。
