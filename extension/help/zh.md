# IPA Reader - 使用指南

> 版本：v1.1.0

## 简介

IPA Reader 是一款专为英语学习者设计的浏览器扩展。它可以在网页上的英文单词旁添加 IPA（国际音标）发音注释，支持美式和英式英语口音，帮助您更轻松地学习英语发音。

---

## 主要功能

- **选中文本标注** — 在网页上选中英文文本，自动显示 IPA 和发音按钮
- **整页 IPA 模式** — 一键为页面所有英文单词添加 IPA 注释
- **美式与英式口音** — 可在美式英语（en-US）和英式英语（en-GB）IPA 之间切换
- **文本转语音** — 点击扬声器按钮收听与所选口音匹配的发音
- **选区朗读** — 选中任意英文文本，点击浮动按钮或右键选择「朗读选区」即可播放
- **悬停提示** — 将鼠标悬停在已标注单词上可查看 IPA 和发音按钮
- **多语言界面** — 支持 38 种界面语言

---

## 使用方法

### 步骤 1：安装扩展

从 [Chrome Web Store](https://chromewebstore.google.com/) 安装 **IPA Reader**，或在开发者模式下本地加载。

### 步骤 2：打开任意网页

访问包含英文内容的任意网页。

### 步骤 3：选中文本或使用浮动按钮

选中需要标注的英文文本，或点击右下角的浮动按钮启用整页 IPA 模式。

### 步骤 4：查看 IPA

将鼠标悬停在单词上可查看 IPA 提示框，点击扬声器图标即可收听发音。

### 步骤 5：朗读选中文本

用鼠标选中任意英文文本，点击浮动扬声器按钮即可朗读；或右键选择「朗读选区」。

> **提示：** 点击浏览器工具栏中的扩展图标可打开设置面板，调整口音类型、语速等。

---

## 选区朗读

选区朗读功能让您选中任意英文文本后一键朗读，非常适合学习句子发音和阅读练习。

**方法 1：浮动按钮**  
用鼠标选中英文文本后，选区右上角会出现扬声器按钮 — 点击即可朗读。

**方法 2：右键菜单**  
选中英文文本后，右键选择菜单中的「朗读选区」。

> **提示：** 选区朗读使用您在设置中配置的句子语速。单词发音使用默认语速。TTS 语音与您选择的口音类型（美式或英式）一致。

## 设置说明

| 设置 | 说明 |
|------|------|
| **启用 IPA** | 总开关，用于启用或禁用 IPA 标注功能 |
| **整页 IPA** | 启用后，为页面所有英文单词显示 IPA（可能影响页面布局） |
| **口音类型** | 选择美式英语或英式英语 IPA 及发音 |
| **句子语速** | 调整句子朗读速度（单词发音不受影响） |
| **悬停提示** | 鼠标悬停时显示 IPA 提示框 |

## 常见问题

**问：为什么在某些页面上无法使用？**  
答：出于安全原因，浏览器扩展无法在特殊页面运行，如 `chrome://`、浏览器设置页或 Chrome Web Store。

**问：某些单词没有 IPA 怎么办？**  
答：IPA 词典涵盖常用英文单词。部分罕见或专业词汇可能没有 IPA 条目。我们会持续扩充词典。

**问：文本转语音没有声音？**  
答：请检查系统音量设置，并确保已安装英语语音包。不同浏览器和操作系统对语音支持各不相同。

**问：整页模式影响页面布局？**  
答：IPA 注释需要额外空间，可能会影响原始页面布局。若影响阅读，可关闭整页模式，改用悬停提示。
